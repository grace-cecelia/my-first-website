Lab 2

January 15, 2026

What I Learned

Coding Basics

    1. When creating multiple pages, use of inline CSS can help keeping the number of CSS classes to a minimum by modifying specific HTML items.

    2. Website layout can be accomplished with a mix of flexbox and grid.

    3. CSS classes can be used to modify multiple elements on a page, while CSS elements should only be used once.

    4. The div tag is an organizational tool that allows you to structure HTML content into containers.

Responsive Design

1. F12 navigates to the DevTools page, which can be used to view the site on different screen sizes.

2. Responsive design varies not only between devices but also between browsers, requiring testing to be done on multiple browsers.

3. The @media query applies styles only on devices with a defined range of screen sizes.

HTML/CSS techniques

1. Drop shadow: implemented on lab 2 images.

2. Replacing header text with an image: to be used to put the header text in my handwriting.

3. Replacing an underline border with an image: to be used to make the underline in the navigation menu appear handwritten.

Reflection

This lab showed me the importance of organization and structure when writing code. While it may seem daunting to create a website, I found that the interplay between HTML and CSS elements made it easy to define elements in a particular way, then apply styles conditionally. It followed the logical structure that if I implemented tags correctly in HTML, I could then implement classes and IDs correctly within CSS. Because many of the skills in this area are widely useful, I found these techniques becoming more intuitive as I applied them elsewhere on the site.

I was also able to use some CSS techniques I picked up last week and through my own curiosity to accomplish tasks beyond the lab activities. These were relatively simple, such as adjusting the padding by sections and elements, but it was nice to figure out small design challenges as I build up to coding my MRP. As I worked on the lab and thought about the ways I want my site to differ from the code in the lab, I looked up a number of tutorials for changes I still wish to make to the site. Having done this lab, I feel capable of following more tutorials and further customizing the site to my liking.
