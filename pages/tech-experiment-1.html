Tech Experiment: Interactive Design

February 4, 2026

Experiment Goals

    1. Build a few simple interactive items in HTML, CSS, and JS as preparation for both my MRP and interactive project.

    2. Learn to code a toggle that shows and hides labels on a click, a dark mode toggle, and a blur filter button.

    3. Explore common development skills needed for interactive design for the web.

Part 1: Toggle for Labels

One element of my MRP is a build-your-own-surgery tool that uses a toggle switch to show and hide labels according to the user's preferences. This experiment involves creating a similar switch to later implement in the final product.

(insert image of prototype switch here)

I used code from W3 Tutorials to create a simple toggle switch, then added text based on code recommended on Stack Overflow. I adjusted CSS elements to produce the appearance I wanted.



I then brought in a sample image to be labeled for the purpose of this experiment. I used a W3 tutorial to create a label on the image with HTML and CSS, then I further stylized the text using CSS.

I then decided to experiment with SVGs as a backing box to the label, so I used HTML to create an SVG rectangle and adjusted the parameters. I then used an image overlay to place the SVG rectangle on top of the image and adjusted the positioning to match that of the label text. (Note for future use: figuring out a way to group the box and label text will streamline this process.)



Hippocampus


The next job was to connect the label appearing and disappearing with the toggle.

I brought in code from W3 that used HTML and JS to hide and show an element with a button. I experimented with adding this code to the items I had created. I wasn't able to immediately connect the label with the toggle, but I implemented JS to connect the hide/show function to the button code provided by W3.

I had some trouble getting this to work properly, as I couldn't figure out how to connect the button function to an item that wasn't a button.



Hippocampus
